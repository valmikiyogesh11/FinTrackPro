This ZIP contains:
- `fintrackpro-backend`: Spring Boot + JPA + MySQL (no security)
- `fintrackpro-frontend`: Next.js + React + Axios

## Backend

1. Import `fintrackpro-backend` as a Maven project.
2. Update `src/main/resources/application.properties` with your MySQL username/password.
3. Create a database named `fintrackpro` in MySQL.
4. Run the application.

## Frontend

1. Open `fintrackpro-frontend` folder.
2. Run:
   ```bash
   npm install
   npm run dev
   ```
3. Ensure backend is running at `http://localhost:8080`.
